\# Mythical Creatures Dataset



A structured dataset of global ghosts, cryptids, monsters, and mythological beings.



\## Structure

Each entry contains:

\- name

\- category

\- origin\_country

\- culture

\- image\_prompt (for AI generation)

\- description

\- latitude

\- longitude



\## Purpose

For AI image generation, GIS mapping, storytelling, and research.
